Questão 07

Usando exclusivamente classes do Bootstrap 5, crie 3 mensagens flutuantes com as seguintes características:

  • A primeira deve ser um Toast com o título "Informação" precedido por um ícone qualquer do Bootstrap Icons, e com o texto "Esta é uma notificação criada pelo componente Toast do Bootstrap 5.".
  • A segunda deve ser um adicionado a um botão intitulado "Posicione o Mouse Aqui". O Tooltip deve aparecer à direita.
  • A terceira mensagem deve ser um adicionado a um botão intitulado "Clique Aqui". O Popover deve aparecer à direita.

* Ao fim do código, há uma tag script contendo o código Javascript para inicialização dos componentes de mensagens. Não apague esse código, pois o funcionamento das mensagens flutuantes depende dele.

Informação
Esta é uma notificação criada pelo componente Toast do Bootstrap 5.
